Knowitpad

The world's most insignificant text editor

Knowitpad is a deliberately small macOS text editor for people who want plain text editing without clutter, feature bloat, or startup lag. It opens quickly, edits locally, and stays out of the way.

What it is for

Knowitpad is meant for the everyday files that do not need a heavyweight IDE or a complex document app. Notes, logs, markdown, snippets, config files, raw text, and odd text-like files all fit naturally here.

  • Create, open, edit, save New files, existing files, Save, Save As, recent files, and native macOS file dialogs.
  • Editing basics that matter Find, replace, go to line, word wrap, adjustable text size, and a status bar with line and column info.
  • Markdown when you want it Markdown files can be read as raw text or viewed in a rendered preview without changing the source.
  • Local-first behavior Files stay on your Mac, settings are remembered, and sessions can be restored between launches.

Capabilities

  • Opens common text files and many unknown text-like formats
  • Can inspect arbitrary files as text when possible
  • Supports light mode and dark mode
  • Remembers window position and size
  • Restores recent work and unsaved sessions
  • Includes spaces-for-tabs behavior when preferred
  • Built to stay lightweight and responsive on macOS

Why it exists

A lot of text editors try to become writing suites, note systems, developer platforms, or cloud products. Knowitpad does not. It is intentionally narrow in scope and is better for it.

The goal is simple: make a dependable editor that feels native on macOS, opens fast, and handles the core text editing workflow cleanly.

Support and privacy

Knowitpad is a local-first app. It does not need an account and does not depend on online services to edit text.
